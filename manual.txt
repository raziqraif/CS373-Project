1. Run Instruction

  This program must be run from the project's root directory.

  Command:
      python source/test.py SVM  (For SVM algorithm)
      python source/test.py KMeans   (For KMeans algorithm)

  After running, the program will generate "out.png" which is the plot of (B or C) and K vs Accuracy.

2. Libraries

  Python version 2.7.15+ was used in this project.

  Additional libraries used were:
      - sklearn
      - numpy
      - matplotlib

  All of the libraries can be installed with pip install <library-name>.
